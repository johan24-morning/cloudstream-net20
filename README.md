# Net20 Cloudstream Provider

Free Net20.cc provider for Cloudstream with TMDB auto-loading.

## Features
- Movies & TV shows
- TMDB integration
- Fast embed loading
- Free only, no premium services

## Installation

1. Upload this repo to GitHub
2. Copy RAW link of repository.json
3. Open Cloudstream
4. Go to Settings → Extensions → Add Repository
5. Paste the RAW link
6. Install Net
